# homebrew-tap

Homebrew formulae that allows installation of the tools published in by Amadeus through the [Homebrew](https://brew.sh/) package manager.


## Installation

```
brew tap AmadeusITGroup/tap
brew install <FORMULA>
```

## Formulae

| Repository | Formula | Description |
| ---------- | ------- | ----------- |
| cdir| [formula](Formula/cdir.rb) | cdir allows you to quickly navigate to a directory recorded into your history |

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
